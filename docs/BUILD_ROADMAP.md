# Build Roadmap

## Milestone 1 — Personal MVP
- Replace in-memory store with PostgreSQL repositories.
- Add PDF/DOCX resume extraction.
- Add encrypted file storage.
- Add candidate correction UI and immutable evidence records.
- Add Ashby connector and employer-career-page ingestion.
- Add search filters and scheduled refresh.

## Milestone 2 — ChatGPT App UI
- Job-card component with match, salary, and gap panels.
- Resume diff viewer.
- Cover-letter editor.
- Approval modal with unresolved screening questions.
- Application pipeline board.

## Milestone 3 — Assisted Apply
- Browser extension with user-controlled field mapping.
- Pause on sensitive or legally meaningful fields.
- Upload approved resume and cover letter.
- Capture confirmation number and status.

## Milestone 4 — Production hardening
- OAuth 2.1 + PKCE for MCP access.
- Per-user row-level security.
- Audit and consent log.
- Prompt-injection isolation for job content.
- Rate limits, retries, observability, evals, and backups.
- Privacy policy, data export, and deletion.
